# ebook-demo
使用epubjs实现简单的阅读器
功能如下：
    1.电子书加载
    2.手指滑动实现翻页
    3.设置字体大小和主题
    4.目录跳转
    5.百分比跳转

## Project setup
```
yarn install
```
